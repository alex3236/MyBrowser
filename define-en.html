MyBrowser

A spam page that provides information about your browser.

The following explanation is partially based on Wikipedia and MDN Web Docs.

Terminology

- Browser: Basic information about the browser. Includes browser name and version number determined by User-Agent, and the operating system information provided by your browser.

- The User-Agent: The User-Agent prefix contains a feature string that allows the counterpart of the network protocol to identify the application type, operating system, software developer, and version number of the user agent software that initiated the request.

- Vendor: The upstream provider reported by the browser.

- View scale: The size of the display area of the browser report.

- Internet Protocol Address: A string of numbers used in the Internet Protocol to identify the device that sent or received the datagram. Provided by IPAPI.co.

- Region: The region you are in, determined by IP address.

About this site

- This is a pure front-end page. The reason for this simplicity is that the entire site load need <100KB of data transport, which makes us to get information quickly in a low-speed network.
